# 📝 Todo App (Fullstack)

Aplicație fullstack pentru gestionarea task-urilor, construită cu:

## 🚀 Tehnologii
- Node.js
- Express
- MongoDB
- JWT Authentication
- HTML, CSS, JavaScript (Frontend)

## 🔐 Funcționalități
- Autentificare utilizator (login/register)
- Creare task
- Vizualizare task-uri
- Update (complete/incomplete)
- Ștergere task

## 📦 API Endpoints

### Auth
- POST /api/auth/register
- POST /api/auth/login

### Tasks
- GET /api/tasks
- POST /api/tasks
- PUT /api/tasks/:id
- DELETE /api/tasks/:id

## 🖥️ Frontend
- Interfață simplă dark mode
- Conectată la backend prin fetch API
